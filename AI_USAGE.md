# AI Usage Document

This document tracks how AI assistant tools were utilized during the creation of SplitSafe.

---

## 1. System Prompt & Instructions
SplitSafe was generated by the **Google DeepMind Advanced Agentic Coding** agent:
- Model used: `Gemini 3.5 Flash` (Medium).
- The agent was tasked to build a complete database schema, folder structure, Prisma models, backend API routes, calculation and settlement engines, custom CSV importer, and React front-end pages.

---

## 2. Agent Contribution Breakdown

### Design Phase
- Designed the relational schemas using Prisma ORM.
- Formulated the date-boundary membership constraint logic.
- Crafted the rounding correction algorithm for Equal, Exact, Percentage, and Share split methods.

### Implementation Phase
- Generated backend TypeScript modules:
  - `currencyService.ts` for conversions.
  - `expenseEngine.ts` for split math.
  - `settlementEngine.ts` for Simplified Debt Settlements.
  - `csvImporter.ts` for RFC-4180 parsing and anomaly flags.
- Built authentication endpoints (JWT validation & Bcrypt hashing).
- Constructed a single-file, highly-responsive React front-end featuring a glassmorphic visual theme.

### Verification Phase
- Created a standalone TypeScript test harness (`engine.test.ts`) that runs assertions without external testing overhead.
- Documented project structure, setup scripts, and Git histories.
